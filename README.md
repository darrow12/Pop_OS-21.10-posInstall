# 🐧 Pop!_OS - Shell Script de pós-instalação

Apenas um Shell Script básico de pós-instalção do Pop_OS para uso pessoal.
<br>
<br>

## 📥 Instalação
```bash
# Mova o arquivo setup.sh para a pasta /home

# Execute o arquivo
bash setup.sh
```
